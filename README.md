# Looped
AI-powered C2C marketplace helping users price and list second-hand items across resale platforms.

## Tech Stack
- React + TypeScript
- Supabase (SQL)
- Claude Vision API
- Vercel

## Features
- AI-generated item descriptions from photos
- Pricing intelligence from competitor listings
- Cross-platform listing support
